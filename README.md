# Projet_SDAA
Projet sur le graphes pour l'UE structure de données et algorithmes (S4 MPCI)

Le code du projet doit être lancé depuis main.py, sinon il y aura des problèmes d'import.
Veiller à dé-commenter les fonctions à lancer, et à modifier les variables adéquates (notamment le chemin du fichier reddit.tsv, et les valeurs pour les tests des algorithmes).
